# 🚀 部署到 GitHub Pages — 操作步骤

## 第一步：注册 GitHub 账号（5分钟）
1. 打开 https://github.com/signup
2. 输入邮箱 → 设置密码 → 设置用户名（记好你的用户名！）
3. 验证邮箱

## 第二步：更新 Git 配置
打开终端，运行：
```bash
git config --global user.name "你的姓名"
git config --global user.email "注册GitHub用的邮箱"
```

## 第三步：创建仓库
1. 打开 https://github.com/new
2. Repository name 填：`highway-design-specs`
3. 选择 **Public**（公开）
4. **不要**勾选 "Add a README file"（我们已经有了）
5. 点击 "Create repository"

## 第四步：推送代码
在终端运行：
```bash
cd "C:\Users\Administrator\Desktop\规范收录网站"
git remote add origin https://github.com/你的用户名/highway-design-specs.git
git branch -M main
git push -u origin main
```

## 第五步：开启 Pages
1. 打开仓库页面 → Settings → Pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main" → / (root) → Save
4. 等 1-2 分钟，你的网站就上线了！

## 🎉 访问地址
```
https://你的用户名.github.io/highway-design-specs/
```
